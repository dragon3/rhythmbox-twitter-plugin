Rhythmbox Twitter Plugin

This is a plugin for Rhythmbox that posts current played song title, album title and author.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/dragon3/rhythmbox-twitter-plugin

Install

Ubuntu 10.04 - Lucid Lynx

1. Install python-simplejson & python-twitter

$ sudo apt-get install python-twitter

2. Install this plugin

$ cd /tmp
$ wget http://github.com/dragon3/rhythmbox-twitter-plugin/tarball/master
$ tar zxfv dragon3-rhythmbox-twitter-plugin-*.tar.gz
$ cd dragon3-rhythmbox-twitter-plugin-*
$ make install

Ubuntu 9.04 - Jaunty Jackalope

1. Install python-simplejson & python-twitter

$ sudo apt-get install python-simplejson
$ cd /tmp
$ wget http://python-twitter.googlecode.com/files/python-twitter-0.6.tar.gz
$ tar zxfv python-twitter-0.6.tar.gz
$ cd python-twitter-0.6
$ python setup.py build
$ sudo python setup.py install

2. Install this plugin

$ cd /tmp
$ wget http://github.com/dragon3/rhythmbox-twitter-plugin/tarball/master
$ tar zxfv dragon3-rhythmbox-twitter-plugin-*.tar.gz
$ cd dragon3-rhythmbox-twitter-plugin-*
$ make install

Setting

1. Enable this plugin.

2. Allow this plugin access to your Twitter account.

3. Copy your PIN.

4. Input your PIN.

If you would like to change your username ...

Open the plugin "Configure" dialog,
and push the button labeled your username.
